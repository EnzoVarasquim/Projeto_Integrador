# Projeto Integrador - Artes Interativas

Projeto desenvolvido utilizando HTML, CSS, Bootstrap e p5.js com o objetivo de criar artes interativas inspiradas em jogos, filmes, animes e temas originais.

## Tecnologias

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- p5.js
- p5.sound

## Artes

- Bloodborne
- Portal 2
- Fullmetal Alchemist
- Outer Wilds
- Project Hail Mary
- Floresta Assombrada
- Prisão
